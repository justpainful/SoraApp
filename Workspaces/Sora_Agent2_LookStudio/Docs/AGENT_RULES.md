# Agent Rules for Sora

You are working on one feature pod only.

You must:
- Build your assigned feature completely.
- Keep changes inside your assigned folders.
- Use the shared core types exactly as written.
- Make the Xcode project build after your changes.
- Report every file you changed.
- Report how to test your feature.

You must not:
- Modify `Sora/App/SoraTypes.swift`.
- Modify `Sora/App/AppState.swift`.
- Modify `Sora/App/SoraPipelineContract.swift`.
- Rename shared protocols or structs.
- Add networking.
- Add 4K60 for v0.1.
- Rewrite another agent's feature.
